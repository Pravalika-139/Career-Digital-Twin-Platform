\# **Career Digital Twin Platform**



\## 📌 **Project Overview**

Career Digital Twin Platform is a web-based application that helps users explore career opportunities based on their skills, interests, and profile information. The platform provides career recommendations and helps users understand suitable career paths.



\## 🚀 **Features**

\- User Profile Management

\- Career Recommendation System

\- Resume Support

\- Interactive Web Interface

\- Career guidance based on user information

\- Project screenshots included



\## 🛠️ **Technologies Used**

\- Python

\- Flask

\- HTML

\- CSS

\- JavaScript

\- Git \& GitHub

\## ▶️ **How to Run the Project**



1\. **Clone the repository:**



&#x20;   git clone https://github.com/Pravalika-139/Career-Digital-Twin-Platform.git



2\. **Open the project folder:**



&#x20;  cd Career-Digital-Twin-Platform



3\. **Install Flask:**



&#x20;  pip install flask



4\. **Run the application:**



&#x20;  python app.py



5\. **Open your browser:**



&#x20;  http://127.0.0.1:5000
